To use the C# SDK you need to add Rabbit and NewtonSoft using NuGet (or bower/npm/etc).

Also - you need to add the following app settings to your app.config. 

       <appSettings>
            <add key="rmq_virtual_host" value="SassyMQHost" />
            <add key="rmq_username" value="guest" />
            <add key="rmq_password" value="guest" />
            <add key="rmq_endpoint" value="sassymq.anabstractlevel.com" />
       </appSettings>
