Public

  1. "Create S M Q Project" to Host in the general channel.

Public can hear

    Owner

    Owner can say

    1. "Take Ownership" to Host in the general channel.
    2. "Set Source" to Host in the general channel.
    3. "Get S D K" to Host in the general channel.
    4. "Rebuild S D K" to Host in the general channel.
    5. "Get Files" to Host in the general channel.
    6. "Get File" to Host in the general channel.
    7. "Get Module Init File Set" to Host in the general channel.

    Owner can hear

      Owner can also hear everything that Public hears.

        5. "Rebuild S D K" from Owner.

        7. "Get File" from Owner.
        8. "Get Module Init File Set" from Owner.
